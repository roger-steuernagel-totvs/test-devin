# Steps

This skill applies only to files that ends with ".p" or ".i" sufixes.
Block codes starts with ":" and ends with "end.".

## For Each Step

When find a "for each" block convert it to java conditional for.

## Display Step

When find a "display" statement convert it to a system out print in java.
